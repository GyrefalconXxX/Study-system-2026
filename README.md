# Study-system-2026
(AI made) A study system, with currency, reward shop, checklists, stopwatch, exp conversion, etc.

_**The code is provided mostly by AI cuz I suck at python lol**_

## Current features:
- stopwatch
- log papers done and in their categorized modules per subject, as well as if it is a {Timed} or {Untimed} paper, as well as if it is a {Tuition} paper or not
  - subjects available are:
    -   **Maths**
    -     P1, P2, P3, P4, S1, S2
    -   **Biology**
    -     U4, U5, U6
    -   **Chemistry**
    -     U4, U5, U6
  - viewing analytics, eg: mean grade

- tracker of theory topics done
  - currently on has S2 and P4 topics

- reward shop
- exp history
- Checklist mode
  - View tasks to do
  - toggle tasks as {Done} {In progress} or {Complete}
  - delete tasks
  - view previously DONE or DELETED tasks and restore as needed
  - Bulk paste a list of tasks when they are in the following format:
    ```
    {Task 1}

    {Task 2}

    {Task 3}

    DONE
    ```
